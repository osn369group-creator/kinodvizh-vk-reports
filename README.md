# Кинодвиж VK Reports

Репозиторий для шаблонов и готовых OSINT-сводок агента **Кинодвиж** по публичным источникам ВКонтакте.

## Назначение

Здесь хранятся:

- шаблоны отчётов;
- журнал проверок;
- карточки событий;
- структура хранения медиа-ссылок;
- правила обезличивания перед публикацией.

## Безопасность

В отчётах нельзя сохранять лишние персональные данные, закрытую информацию, приватные сообщения, токены, cookie, адреса частных лиц или данные для преследования.

## Файлы

- `templates/vk-osint-brief.md` — короткая сводка.
- `templates/vk-incident-report.md` — подробный отчёт по событию.
- `templates/media-log.md` — журнал фото/видео.
- `logs/research-log-template.md` — журнал поиска.
